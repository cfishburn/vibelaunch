The member route demonstrates the scaffold's BFF pattern.

- Clerk protects the page before Astro renders it.
- The React island fetches its data from an Astro API route, not directly from Clerk's backend.
- The browser never needs direct access to any server-side service.

This is the pattern the AI guardrails document as the correct way to add protected surfaces. Fork the scaffold, change `src/config/site.ts`, and build from here.
