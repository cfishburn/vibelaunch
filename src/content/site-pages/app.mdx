This member route is a reference implementation, not product functionality.

- Clerk protects the page before Astro renders it.
- The React island fetches its data from an Astro API route, not directly from Clerk's backend.
- The browser never needs direct access to any server-side service.

Fork the scaffold, replace the branding in `src/config/site.ts`, and use this pattern when you add your own protected surfaces.
