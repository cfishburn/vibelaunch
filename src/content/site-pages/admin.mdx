This page is the smallest useful admin slice in the starter.

- The route itself is gated in middleware by `createRouteMatcher`.
- The JSON endpoint behind it is also gated server-side.
- Admin access is driven by the `CLERK_ADMIN_USER_IDS` allowlist in `.env`.

The guard suite (`guard-secrets`) ensures that allowlist values never get hardcoded in source. The AI guardrails (`CLAUDE.md`) tell assistants to use env vars instead.
